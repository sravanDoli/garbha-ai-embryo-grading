# Garbha AI – Embryo Grading & Tracking System

## Overview
Garbha AI is an AI-powered system designed to assist embryologists in analyzing and grading embryo development using computer vision techniques. The system automates embryo fragmentation detection and integrates with a backend system for tracking, reporting, and management.

## Problem Statement
Manual embryo grading in IVF labs is subjective, time-consuming, and prone to inconsistencies. This project provides an automated and objective AI-based solution.

## Solution
- YOLO-based embryo fragmentation detection
- FastAPI backend for tracking and management
- PostgreSQL database integration
- Automated prediction and visualization pipeline

## Tech Stack
- Python, OpenCV, NumPy, Pandas  
- YOLOv8 (Ultralytics)  
- FastAPI, SQLAlchemy  
- PostgreSQL
